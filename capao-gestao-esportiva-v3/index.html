Gerencie campeonatos, inscrições, clubes, confrontos, resultados, artilharia, denúncias e punições sem misturar as informações da parte pública.

Criar campeonato

Categorias e séries do campeonato

Configure primeiro a categoria do campeonato. Depois, se necessário, ative as séries dentro dessas categorias.

Categoria é a divisão principal, por exemplo: Masculino, Feminino, Sub-17 ou Veterano.
Use este campo somente se escolher Personalizado ou quiser adicionar categorias extras.
Séries são divisões internas, como Série Ouro, Série Prata e Série Bronze.
Ex.: Masculino com Ouro/Prata/Bronze e Feminino com série única.
Use apenas quando selecionar “Escolher manualmente”.
O sistema apenas sugere. A confirmação final sempre fica com o administrador.

Regras dos campeonatos

Cadastre manualmente as regras de cada campeonato. Você também pode ativar ou desativar a aplicação subsidiária da CBJD por campeonato.

Texto padrão recomendado: Em caso de omissão, dúvida interpretativa ou ausência de regra específica no regulamento municipal, será aplicada subsidiariamente a CBJD, quando ativada para este campeonato.

Base de consulta CBJD

Resumo de referência para consulta rápida. Para decisões oficiais, consulte sempre o texto integral oficial.
